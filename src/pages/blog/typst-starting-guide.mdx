Code highlighting Typst supports for code highlighting is pretty good you can use the same syntax as markdown.

<Typst
inject={`
#set page(
  width: auto,
  height: auto,
  numbering: "1",
)
`}

\`\`\`typ

\`\`\`

Typst also allows you to create function and have state/variables you can use these to create a table of pricing for example and then calculate the end result.

<Typst
inject={`
#set page(
  width: auto,
  height: auto,
)
`}
  code={`
#import "@preview/oxifmt:0.2.1": strfmt
#let num = state("total", 0.0)

#let add_price(price) = {
  return [\\$#strfmt("{:.2}", price) #num.update(c=> c+price)]
}

#let format_price() = context {
  return strfmt("\${:.2}", num.get())
}

#table(
  columns: 2,
  [*Name*], [*Price*],
  [Cat Food], add_price(2.2),
  [Dog Food], add_price(2.2),
  [Rocket!], add_price(400),
  table.cell(colspan: 2, fill: gray)[],
  [Total], format_price()
)

`}
  name="math"
/>
